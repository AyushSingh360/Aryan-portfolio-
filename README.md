## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ and npm/pnpm
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/AyushSingh360/Aryan-portfolio-c8.git
cd Aryan-portfolio-c8

# Install dependencies
npm install
# or
pnpm install
```

### Development

```bash
# Start development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
# Create optimized production build
npm run build

# Start production server
npm start
```

## ğŸ¨ Customization

To customize this portfolio for your own use:

1. **Personal Information**: Update `app/layout.tsx` for metadata
2. **Contact Details**: Edit `components/contact.tsx`
3. **Resume**: Replace `public/Resume.pdf` with your resume
4. **Experience**: Modify `components/experience.tsx` with your work history
5. **Projects**: Update `components/projects.tsx` with your projects
6. **Skills**: Change `components/tech-stack.tsx` for your tech stack
7. **Colors**: Adjust theme in `tailwind.config.ts` and `app/globals.css`

## ï¿½ Contact Form Setup (Optional)

The portfolio includes a contact form. To enable email functionality:

1. Sign up for [Resend](https://resend.com)
2. Get your API key
3. Add to `.env.local`:
   ```env
   RESEND_API_KEY=your_api_key_here
   ```
4. Uncomment Resend integration in `app/api/contact/route.ts`
